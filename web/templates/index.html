О сайте

Этот сайт был создан для телеграм бота "Студсовет СПбГАСУ", с помощью него, админам предоставляется функционал добавления и изменения информации в боте.


Краткий экскурс по функционалу:

1. Пользователи:

Даёт возможность посмотреть аккаунты, которые запускали бота, для каждого пользователя выводится дата активации бота и ссылка на его телеграмм.

2. FAQ:

Позволяет добавлять разделы, вопросы и ответы на них в пункт телеграм бота FAQ, помимо этого все записанные и существующие данные хранятся в виде таблицы, которую можно удалять и заполнять.

3. Управление администраторами:

Этот раздел даёт возможность добавлять и исключать админов телеграм бота, даются два поля ввода: Телеграм id и Имя Админа, записанные данные сохраняются и выводятся в виде таблицы.

4. Мероприятия:

С помощью этого пункта, админы могут добавлять даты и названия ближайших мероприятий для информирования пользователей телеграм бота, данные сохраняются в виде таблицы и могут удаляться.
